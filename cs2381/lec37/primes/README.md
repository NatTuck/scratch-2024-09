# Notes on Cache

Old laptop: 8th gen Intel i7 processor (~ 2.0 GHz)

    - RAM: 16 GB
      - Latency: 60 ns = 120 cycles
      - Throughput: 20 GB/s
    - L3 cache: 
      - 8 MB (shared)
      - Latency: 11ns = 22 cycles
      - Throughput: > 200 GB/s
    - L2 Cache:
      - Size: 256 KB (per core)
      - Latency: 3ns = 6 cycles
      - Throughput: > 400 GB/s
    - L1 Data Cache:
      - Size: 32k (per core)
      - Latency: 1ns = 2 cycles
      - Throughput: > 1000 GB/s
